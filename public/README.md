# Pasta `public/`

Assets estáticos servidos diretamente pelo Next.js.

## Conteúdo previsto

- `favicon.ico`, `favicon-16x16.png`, `favicon-32x32.png`
- `apple-touch-icon.png`, `android-chrome-192x192.png`, `android-chrome-512x512.png`
- `logo.svg`, `logo-light.svg`, `logo-dark.svg`
- `og-image.png` (Open Graph padrão)
- `templates/` — screenshots e previews dos templates de currículo
- `blog/` — imagens de capas para o blog
- `illustrations/` — ilustrações de empty states
- `fonts/` — fontes auto-hospedadas (opcional)

> Referenciados via `/caminho/arquivo.png` no código.
