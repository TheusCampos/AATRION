# Pasta `styles/`

Estilos globais, tokens de design e configuração do Tailwind.

## Estrutura prevista

```
styles/
├── globals.css               # Reset, Tailwind base, variáveis CSS
├── tokens.css                # Design tokens (cores, espaçamentos, tipografia)
└── animations.css            # Keyframes compartilhados
```

> Os tokens em [`/docs/architecture/design-tokens.md`](../docs/architecture/design-system.md)
> devem ser espelhados nas variáveis CSS desta pasta.
